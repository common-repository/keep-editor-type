== Description ==


TinyMCEのビジュアルエディタ or テキストエディタのエディタタイプを投稿毎に変更します。

これにより、ビジュアルエディタとテキストエディタを行き来することなく、
・ビジュアルエディタで入力した投稿はビジュアルエディタで開く
・テキストエディタで入力した投稿はテキストエディタで開く
ことができます。


デフォルトのEditorのみサポートしています。

This plugin change TinyMCE's editor type(visual editor or text editor) for each post, not for each user.
Only the default Editor is supported.


* [サポートサイト](https://labo.hanabanadesign.com/keep-editor-type/)



## Introduction ##

### Features ###

* TinyMCEのエディタタイプを投稿毎に変更します。
（投稿毎に最終エディタタイプを保持します。）



### How to use ###

1. Install & Activate the plugin.

プラグインを有効化するだけです。
あとは投稿の保存時に最終のエディタタイプが保存されます。
次回その投稿を開く時に、その投稿の最終エディタタイプで開きます。



### 注 ###

必ず、バックアップなどをした上で、使用してください。
既に、エディタのカスタマイズなどを使用されている場合、動作しない可能性があります。

本プラグインを使用した事による、いかなる損害も作者は一切の責任を負いませんので、自己責任の上でご使用下さい。







== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `keep-editor-type.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates


#### WP管理画面から ####
1. 「プラグイン ＞ 新規追加」 から「Keep Editor Type」を探します。
2. 「今すぐインストール」をクリックして、インストールします。
3. プラグインを有効にします。
